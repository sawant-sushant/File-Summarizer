# AI Document Summarizer 📝🤖

A simple, lightweight Python script that uses Google's Gemini AI to automatically read text documents and generate concise, 3-point summaries.

## 🚀 Features
* **Automated Summarization:** Instantly boils down long text files into easy-to-read bullet points.
* **Locally Executed:** Runs entirely from your terminal.
* **Powered by Gemini:** Uses the fast and intelligent `gemini-2.5-flash` model.
